Variabelen

Hoe maak je een variabele aan?

Expressies

Elk programma maakt bewerkingen met data. In dit hoofdstuk leer je de meest gekende bewerkingen toepassen binnen een programmeertaal.

Arrays

Lijsten gebruiken en aanpassen.

Instructies

Programmeren is niet meer dan het geven van bevelen aan een computer. Zo'n bevel heet een instructie.

BELANGRIJKE TERMEN

Variabelen
variabele Een opslagplaats met een naam, waarin je een waarde kunt opslaan.
datatype De soort, type of categorie van een waarde.
declaratie Het aanmaken van een variabele.
initialiseren Het geven van een eerste waarde aan een variabele, wanneer die voor het eerst wordt aangemaakt.
waarde Eender wat door een computer kan worden opgeslagen.
Expressies
expressie Een bewerking met als resultaat een waarde.
rekenkundige operatie Een wiskundige bewerking tussen twee getallen.
logische operatie Een logische bewerking tussen twee waardes.
operatie De verschillende soorten bewerkingen die een programmeur kan gebruiken.
voorwaarde Een bewerking met als resultaat een booleaanse waarde.
operator Het teken van een bewerking.
Arrays
index Een getal of andere waarde die aanduidt op welke plaats een waarde zich bevindt in de lijst.
indexed array Een lijst waarbij de index een automatisch oplopend getal is. In de meeste programmeertalen begint deze index bij 0 te tellen.
associative array Een lijst waarbij de index een zelfgekozen waarde is.
Instructies
instructie Een bevel aan een computer dat meestal bestaat uit het opslaan of aanpassen van data.
commentaar stukken tekst die je toevoegen aan je code die genegeerd wordt door de computer. Dit kan gebruikt worden om wat extra uitleg te geven aan moeilijke code.
assignment Oefen theorie
print Print deze pagina bug_report Meld een fout create Bewerk deze pagina
