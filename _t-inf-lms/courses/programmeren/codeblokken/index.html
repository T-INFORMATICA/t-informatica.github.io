Wat zijn codeblokken?

Functies

Functies zijn één van de belangrijkste codeblokken om te leren gebruiken. Ze worden gebruikt om je code beter te verdelen en structureren.

BELANGRIJKE TERMEN

Codeblokken
codeblok Een lijst met instructies dat begint en eindigt met een bepaald teken.
functie Een lijst met instructies dat begint en eindigt met een bepaald teken waar je een naam aan geeft. De instructies worden uitgevoerd als je de naam aanroept.
selectie Een lijst met instructies dat enkel wordt uitgevoerd wanneer een voorwaarde waar is.
iteratie Een lijst met instructies dat verschillende keren achter elkaar wordt uitgevoerd.
Functies
signature De unieke naam van een functie. Bij veel programmeertalen horen de parameters hier ook bij.
return value De waarde die een functie teruggeeft aan de plaats waar de functie werd aangeroepen.
parameter Een waarde die aan een functie wordt meegegeven.
Selectie
sequentie Het uitvoeren van instructies in de volgorde dat ze geschreven zijn.
selectie Het eenmalig uitvoeren van code wanneer een voorwaarde waar is.
Iteratie
itereren Het herhaaldelijk uitvoeren van code zo lang een voorwaarde waar is.
assignment Oefen theorie
print Print deze pagina bug_report Meld een fout create Bewerk deze pagina
